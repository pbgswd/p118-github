## Project Title: Local 118 Rebuild

### Background

IATSE Local 118 in Vancouver has a website which has served its needs well in the past. Now a new instance of the
website that makes use of current software paradigms will better serve the Local. It will allow for management of more timely
information, addition of online services and the reorganization of current ones to best suit the needs of the Local membership and public visitors of the website.

### High-Level Project History

Rebuild the site in current software in a development environment, making it possible to demonstrate the site offline.
The site news content will be copied in and reorganized. 

Collaboration and input from the client and stakeholders was solicited during development, however, the developer took the 
lead to build out the website with the framework for data schema, authentication, privileges, content management and administration,
and layout. A Waterfall/Agile project management methodology was used in the accepted project plan.

The second phase of the project will import member data and content, build out improvements for the site and plan the
user migration to the new site, change to the new site and retire the current website.

The site was delivered May 1, 2020.
